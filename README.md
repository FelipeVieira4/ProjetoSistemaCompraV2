## Criar Banco Dados da aplicação

### Windows
Basta apenas executar o arquivo commando_sql.sql no MySQL Workbench

### Linux
Execute o commando

#### MariaDB
sudo sql < comando_sql.sql

#### MariaDB
sudo mariadb < comando_sql.sql
